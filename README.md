# Sistema de Inventario TI

Proyecto desarrollado para el curso Construcción de Software.

## Descripción
Sistema de inventario desarrollado en Python que permite:

- Agregar equipos
- Mostrar equipos
- Buscar equipos
- Cambiar estado de equipos
- Eliminar equipos

La información se almacena en un archivo JSON.

## Tecnologías utilizadas

- Python
- Git
- GitHub
- GitHub Actions
- GitFlow

## Flujo GitFlow implementado

- main
- develop
- feature/reportes
- feature/validaciones
- release/v1.0
- hotfix/correccion-menu
